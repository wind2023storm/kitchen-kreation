# KitchenKreation - 2D/3D Javascript Kitchen Planner

#### To run and develop locally a HTTP server needs to be intialised;

Either npm or Python are valid options

###### npm

`npm install http-server -g`
`http-server . -p 8000`

###### python

`python -m http.server`
